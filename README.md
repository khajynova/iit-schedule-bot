# 🎓 Бот расписания ИИТ БГУИР

Telegram-бот для отслеживания расписания преподавателей и групп ИИТ БГУИР с интеграцией в Google Календарь.

## 📌 Возможности

- 📅 Просмотр расписания на сегодня, завтра, неделю, месяц
- 🔔 Автоматические уведомления об изменениях в расписании
- 👨‍🏫 Поддержка преподавателей и групп
- 📊 Статистика по занятиям
- ⚙️ Настройка уведомлений
- 🎯 Фильтрация занятий для выбранного преподавателя или группы
- 📅 Интеграция с Google Календарем (ICS-ссылка)

## 🚀 Быстрый старт

### Для пользователей

1. Найдите бота в Telegram: `@ваш_username_бота`
2. Отправьте команду `/start`
3. Установите преподавателя или группу:
   - `/set_teacher Хаджинова Н.В.` — для преподавателя
   - `/set_group 60131` — для группы
4. Готово! Используйте кнопки или команды для просмотра расписания

#### 📋 Команды бота

| Команда                   | Описание                                                           |
| ------------------------- | ------------------------------------------------------------------ |
| `/start`                  | Главное меню                                                       |
| `/set_teacher ФИО`        | Установить преподавателя (например, `/set_teacher Хаджинова Н.В.`) |
| `/set_group Номер_группы` | Установить группу (например, `/set_group 60131`)                   |
| `/today`                  | Расписание на сегодня                                              |
| `/tomorrow`               | Расписание на завтра                                               |
| `/week`                   | Расписание на неделю                                               |
| `/settings`               | Настройки бота                                                     |
| `/remove_teacher`         | Удалить преподавателя/группу                                       |
| `/help`                   | Помощь                                                             |

### 📅 Интеграция с Google Календарем

После установки преподавателя или группы вы можете добавить расписание в Google Календарь:

1. Нажмите кнопку **"📅 Google Календарь"** в главном меню
2. Скопируйте полученную ссылку
3. В Google Календаре: **Настройки → Добавить по URL**
4. Вставьте ссылку и нажмите **"Добавить календарь"**

**Ссылки для быстрого доступа:**

- Для группы: `https://iit-schedule-bot.onrender.com/schedule/60131.ics`
- Для преподавателя: `https://iit-schedule-bot.onrender.com/schedule/Хаджинова%20Н.В..ics`

### Для разработчиков (локальный запуск)

#### Требования

- Python 3.10+
- Git
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

#### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/khajynova/iit-schedule-bot.git
cd iit-schedule-bot

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Создайте файл .env с токеном
echo TELEGRAM_BOT_TOKEN=ваш_токен_здесь > .env

# 6. Запустите бота
python bot.py
```

🐛 Решение проблем
Бот не запускается
bash

# Проверьте установку зависимостей

pip install -r requirements.txt

# Проверьте наличие токена в .env файле

cat .env

# Проверьте логи в Render (вкладка "Logs")

Конфликт при запуске (Conflict)
Ошибка Conflict: terminated by other getUpdates request означает, что бот уже запущен где-то еще. Решение:

Остановите локального бота (закройте терминал)

На Render добавлен автоматический сброс вебхука при запуске

Бот не отвечает
Проверьте, запущен ли бот на Render (вкладка "Logs")

Проверьте, не истек ли токен (получите новый у @BotFather)

Проверьте, активен ли keep-alive (вкладка "Logs" покажет пинги)

База данных не создается

# Создайте папку вручную

mkdir data
chmod 755 data # На Linux/Mac
Google Календарь не показывает события
Импортируйте ICS-файл вручную через Настройки → Импорт и экспорт

Дождитесь автоматической синхронизации (до 12 часов)

Удалите и добавьте календарь заново
